# Tech Stack

## Backend

- **Node.js**
- **Express.js**
- **MongoDB** (Local for development, MongoDB Atlas for deployment)
- **Mongoose**

## File Upload & Validation

- **Multer** – CSV file upload handling
- **csv-parse** – CSV parsing
- **Zod** – Request and data validation

## Utilities

- **Day.js** – Date and time handling
- **dotenv** – Environment variable management
- **cors** – Cross-origin resource sharing

## Development Tools

- **Nodemon** – Development server auto-reload
- **MongoDB Compass** – Database GUI for local development
